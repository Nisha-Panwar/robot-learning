# 🤖 Robot Learning — Robot Framework

This is my learning project for Robot Framework automation testing.

## 📁 Project Structure
```
robot-learning/
└── tests/
    ├── my_first_test.robot   # Basic Robot Framework tests
    └── login_test.robot      # OrangeHRM login automation
```

## ✅ Tests Covered
- Basic test cases (variables, logging, math)
- OrangeHRM login automation using SeleniumLibrary

## ⚙️ Setup
1. Install Python 3.14+
2. Install dependencies:
pip3 install robotframework
pip3 install robotframework-seleniumlibrary
pip3 install webdriver-manager

## ▶️ Run Tests
robot tests/my_first_test.robot
robot tests/login_test.robot
